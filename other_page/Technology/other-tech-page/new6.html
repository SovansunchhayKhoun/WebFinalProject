ទីបំផុត Google Chrome បានបង្ហើបអំពីមុខងារកាត់បន្ថយ RAM ហើយ

12 ធ្នូ 2022
ចំនួនមតិ 0 | ចំនួនចែករំលែក 0

ជាច្រើនឆ្នាំមកហើយ ដែលអ្នកប្រើប្រាស់បានចាប់អារម្មណ៍ថា Google Chrome ជាកម្មវិធីត្រូវការប្រើប្រាស់ RAM អស់ច្រើនជាងគេ បើប្រៀបធៀបទៅនឹងកម្មវិធី Browser ដទៃទៀត។ ហេតុនេះហើយ Google ក៏កំពុងតែចាប់ផ្ដើមធ្វើតេស្ដលើមុខងារថ្មីមួយ ដែលមានឈ្មោះហៅថា Memory Saver ដើម្បីកាត់បន្ថយការប្រើប្រាស់ RAM សម្រាប់អ្នកប្រើប្រាស់កុំព្យូទ័រជាមួយ RAM ចំនួនតិច។

Memory Saver ជាមុខងារសម្រាប់ឲ្យអ្នកប្រើប្រាស់អាចបិទនិងបើកបាន នៅលើកម្មវិធី Google Chrome សម្រាប់ Windows, macOS និង ChromeOS ទាំងអស់។ មុខងារនេះនឹងធ្វើការផ្អាកដំណើរការលើផ្ទាំងវេបសាយ Tab ដែលអ្នកប្រើប្រាស់បានទុកចោលយូរជាងគេ ដើម្បីកាត់បន្ថយការប្រើប្រាស់ RAM ដូច្នេះទាល់តែអ្នកចុចលើផ្ទាំងវេបសាយនោះ ទើបបានវានឹង Reload វេបសាយឡើងវិញម្ដងទៀត។

ជាការពិតណាស់ Google បានសន្យាថា គ្រាន់តែបើកមុខងារថ្មីនេះ ការប្រើប្រាស់ចំនួន RAM នឹងអាចកាត់បន្ថយបានដល់ទៅ 40% តែម្ដង។ លើសពីនេះ មុខងារ Energy Saver ក៏កំពុងតែស្ថិតក្នុងការសាកល្បងប្រើដូចគ្នា ដែលជាមុខងារជួយសន្សំថាមពលថ្ម សម្រាប់អ្នកប្រើប្រាស់នៅលើកុំព្យូទ័រយួរដៃតែម្ដង។

Google បានប្រាប់ថា មុខងារ Memory Saver និងមុខងារ Energy Saver ថ្មីនេះ នឹងចាប់ផ្ដើមដាក់ឲ្យប្រើប្រាស់ នៅក្នុងរយៈពេលប៉ុន្មានសប្ដាហ៍ខាងមុខនេះ។
